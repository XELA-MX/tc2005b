
## Como hacer un `Script`

* Usando un archivo externo

```HTML

<script type = "text/javascript" src = "codigo.js"></script>

```


- En el mismo documento

```HTML

◦<script type = "text/javascript" >

/*Código javascript*/

◦</script>

```


## Tipos Primitivos

- Número
	- Precisión de `double`
- `String`
	- Se pueden representar dentro de comillas simples y dobles. ('' , "").
- `Boolean`
	- `true y false`
- `Undefined`
- `Null`
